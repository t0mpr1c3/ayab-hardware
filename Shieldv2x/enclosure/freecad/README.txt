# Summary

I have designed a cover for the Brother KH-910 Knitting machine.
You can mount an Adruino Uno  and the AYAB shield in the cover.
There are 2 versions, a stable version and a thin-walled version.

Information about the AYAB project can be found at http://ayab-knitting.com/

In addition, you need a USB extension cable like this one:

https://www.startech.com/Cables/USB-2.0/Internal-and-Panel-Mount/1-ft-Panel-Mount-USB-Cable-B-to-B-Female-to-Male~USBPNLBFBM1
